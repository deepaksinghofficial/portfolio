# Deepak Singh - Software Tester Portfolio

A modern, responsive portfolio website for Deepak Singh, a Software Tester at Liangtuang Technology Private Limited.

## 📋 Project Overview

This is a one-page portfolio website built with:
- HTML5
- CSS3
- MDB Bootstrap Framework
- Custom light pink theme

## 🎨 Features

- Modern and clean design with a light pink color scheme
- Fully responsive layout for all device sizes
- Interactive UI elements with smooth animations
- SEO optimized with meta tags and structured data
- Fast loading and efficient code

## 📁 Project Structure

```
/deepak/
│── index.html              # One-Page Website
│── assets/
│   ├── css/
│   │   ├── custom.css      # Custom CSS file (Light Pink Theme)
│   ├── images/             # Profile picture, project images
│── favicon.ico             # Website Icon
│── README.md               # Project Documentation
```

## 🚀 Sections

1. **Hero/Introduction** - Brief introduction with profile image
2. **About** - Personal information and key skills
3. **Skills** - Technical skills and proficiency levels
4. **Experience** - Work history and achievements
5. **Projects** - Portfolio of testing projects
6. **Contact** - Contact form and personal information

## 🛠️ Setup Instructions

1. Clone the repository:
```
git clone https://github.com/yourusername/deepak-portfolio.git
```

2. Open `index.html` in your browser to view the website.

3. To make changes:
   - Modify `index.html` for content changes
   - Edit `assets/css/custom.css` for styling changes
   - Add images to `assets/images/` directory
